# Advent Of Code 2019

Please, refer to the main site: https://adventofcode.com/2019

## Compile

Use the command:

```
make compile
```

## Run

Use the command:

```
make run DAY={DAY} PART={PART} INPUT={TEXT_FILE}
```

Where
 - `DAY` is the number of the day
 - `PART` is the part of the challenge (1, 2)
 - `INPUT` is your input files
